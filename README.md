🐺 Kenshi Translator & Mod Manager v1.3
I was tired of tons of mods, patches for patches, and endless compatibility issues.  
After a couple of “mental breakdowns” I told myself: enough — I need a proper tool.  
That’s how Kenshi Translator & Mod Manager was born.  

Originally it was just a translator, but then I thought: “Why not make it a full-fledged mod manager as well?”  

✨ What’s New in v1.3

- 🟡 Unified notifications: now every module has a “foolproof” popup (yellow border) if the mod list is missing or a scan hasn’t been run.  
- 🗑️ Clear Mods — the button now properly clears `mods_all.json` and shows a confirmation message.  
- 🔍 Scan Mods — improved scanner: filters junk authors, extracts links and dependencies. If settings or mods are missing, a popup is shown instead of silent exit.  
- 📑 Sorting Mods — added safety check: if `mods_all.json` is empty or missing, a “PRESS SCAN MOD :)” popup appears.  
- 📂 Profiles — save and load up to 3 mod profiles (enabled/disabled). Easily switch between different mod setups.  
- ⚙️ Load/Save Config — load and save `mods.cfg`. When saving, a ZIP backup is automatically created (keeps the last 5).  
- 🖼️ Side Panels for convenience:  
  - **My Game** (cfg + profiles)  
  - **Translate** (translator and language selection)  
  - **Info** (right-click a mod → shows author, language, Steam ID, links, dependencies).  
- 🎨 Interface:  
  - dark clean style, side buttons, mod tree with icons (enabled/disabled, warnings),  
  - drag&drop removed as unnecessary, double-click toggles status,  
  - right-click opens info panel.  
- 🕹️ Start Kenshi — launch the game directly from the manager (supports both Steam and GOG versions).  
- 🚪 Exit — clean program shutdown.  

🔧 Installation (v1.3)
- 1. Download the program archive.  
- 2. Extract it into your game folder Kenshi\mods  
- 3. Go to Kenshi\mods\KenshiTranslator\prog\ and run KenshiTranslator.exe.  
- 4. KenshiTranslator will create a shortcut on the desktop.  

🛡️ Protection
- Checks correct installation (must be in `KenshiTranslator\prog`).  
- Verifies Steam Workshop and game folder paths.  
- If launched from the wrong place → red popup “INSTALL TO CORRECT LOCATION!!!”.  

⚠️ Important
- The program is built with PyInstaller and packed with UPX, so antivirus false positives are possible.  
- The source code is open — anyone can verify it.  

📌 In Development
- Translations: full integration of mod translations directly into the manager.  

⚖️ Kenshi Translator & Mod Manager v1.3 — now a full-fledged tool for managing mods:  
clean interface, sorting, profiles, CFG handling, quick game launch, and foolproof error protection.

✨ Новое в версии 1.3

- 🟡 Унифицированные уведомления: теперь в каждом модуле есть «защита от дураков» с всплывающим окном (жёлтая рамка), если не найден список модов или не выполнено сканирование.  
- 🗑️ Clear Mods — кнопка теперь корректно очищает список (`mods_all.json`) и показывает сообщение об удалении.  
- 🔍 Scan Mods — улучшен сканер: фильтрация мусора в авторах, ссылки, зависимости. При отсутствии настроек или модов теперь выводится окно, а не немой выход.  
- 📑 Sorting Mods — добавлена проверка и защита: если `mods_all.json` пуст или отсутствует, выводится сообщение «PRESS SCAN MOD :)».  
- 📂 Profiles — сохранение и загрузка профилей (3 слота) с модами (вкл/выкл). Профиль сохраняет состояние, легко переключать сборки.  
- ⚙️ Load/Save Config — загрузка и сохранение `mods.cfg`. При сохранении автоматически создаётся ZIP-бэкап (хранится 5 последних).  
- 🖼️ Панели — боковые панели для удобства:  
  - **My Game** (cfg + профили)  
  - **Translate** (выбор переводчика и языков)  
  - **Info** (ПКМ по моду → показывает автора, язык, Steam ID, ссылки, зависимости).  
- 🎨 Интерфейс:  
  - тёмный стиль, кнопки в панелях, дерево модов с иконками (вкл/выкл, предупреждения),  
  - drag&drop удален за ненадобностью, двойной клик — переключение статуса,  
  - ПКМ — просмотр информации.  
- 🕹️ Start Kenshi — запуск игры прямо из менеджера (поддержка Steam и GOG версий).  
- 🚪 Exit — корректный выход из программы.  

🔧 Установка (версия 1.3)

- 2. Распакуйте его в папку вашей игры Kenshi\mods
