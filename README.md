üîß Installation (version 1.2)  
1. Download the archive with the mod.  
2. Extract it into your game‚Äôs folder:  
   Kenshi\mods\  
3. As a result, you should have this folder:  
   Kenshi\mods\KenshiTranslator\prog\  
4. Go to the prog folder and run:  
   KT-Setup.exe  
5. The installer will copy the files into the Kenshi folder, create a desktop shortcut, and finally show ‚ÄúInstalled‚Äù.  
6. After finishing, the installer will delete itself.  

üõ°Ô∏è About antivirus  
If your antivirus starts complaining ‚Äî don‚Äôt panic. This is a false positive.  
Why:  
- the installer creates a shortcut and deletes temporary files (normal behavior, but antiviruses love to call it a ‚ÄúTrojan‚Äù);  
- the program is built with PyInstaller ‚Äî such exe files are almost always flagged as ‚Äúunknown‚Äù;  
- additionally, UPX-packing is used (compression of exe files). Antiviruses hate it, and on VirusTotal half of them ‚Äúgo crazy‚Äù right away. In reality, it‚Äôs just packing that reduces file size and speeds up startup;  
- there is no virus here, it‚Äôs just a mod manager for Kenshi with translation features.

‚úÖ The program is clean and safe.
